> 实验日期：2026-05-30

---

## 2. 处理流程

```text
图像读取
-> resize_max_side(900)
-> GaussianBlur(5)
-> CLAHE 光照归一化
-> HSV 颜色空间转换
-> green / yellow / brown / dark 四通道 mask 生成
-> 形态学开闭运算(kernel=7)
-> 最大连通区域提取(min_area_ratio=0.01)
-> HSV 颜色比例特征 + GLCM 纹理特征提取(levels=32, distances=[1, 2])
-> 人工规则分类
-> 输出 summary.csv / combined_features.csv / 可视化结果
```

## 3. 特征分布统计

### 3.1 ripe 类别（n=20）

| 特征 | 均值 | 标准差 | 最小值 | Q25 | 中位数 | Q75 | 最大值 |
|------|------|--------|--------|-----|--------|-----|--------|
| green_ratio | 0.0151 | 0.0385 | 0.0000 | 0.0000 | 0.0009 | 0.0078 | 0.1327 |
| yellow_ratio | 0.8126 | 0.1140 | 0.5019 | 0.7368 | 0.8311 | 0.8978 | 0.9563 |
| dark_ratio | 0.3592 | 0.1431 | 0.1720 | 0.2812 | 0.3124 | 0.4403 | 0.6683 |
| glcm_contrast | 0.9028 | 0.3442 | 0.3660 | 0.7218 | 0.8796 | 0.9825 | 1.7382 |
| glcm_homogeneity | 0.8926 | 0.0387 | 0.8206 | 0.8675 | 0.9010 | 0.9237 | 0.9473 |
| mask_area | 242220 | 113330 | 80126 | 152266 | 220390 | 321991 | 477912 |

### 3.2 关键观察

1. `green_ratio` 极低，说明该特征对区分未成熟香蕉具有潜在价值。
2. `dark_ratio` 波动较大，茎端、尖端、阴影和表面斑点都会被计入暗色区域。
3. 32 级量化下 `glcm_contrast` 实测范围约为 0.37 - 1.74，原始阈值 35.0 不适用于当前实现。
4. `glcm_homogeneity` 集中在较高区间，说明当前成熟香蕉样本整体纹理较均匀。

## 5. 分类结果

### 5.1 当前阈值配置

```yaml
rules:
  green_ratio_threshold: 0.30
  dark_ratio_threshold: 0.40
  contrast_threshold: 1.45
  yellow_ratio_threshold: 0.50
  dark_ratio_low: 0.32
  overripe_yellow_max: 0.55
```

### 5.2 规则逻辑

1. `green_ratio >= 0.30` -> `unripe`
2. `dark_ratio >= 0.40 and yellow_ratio <= 0.55` -> `overripe`
3. `glcm_contrast >= 1.45 and dark_ratio >= 0.32` -> `overripe`
4. `yellow_ratio >= 0.50` -> `ripe`
5. fallback -> `ripe`

### 5.3 准确率

| 指标 | 数值 |
|------|------|
| 正确分类 | 19 / 20 |
| 准确率 | 95.00% |
| 误判为 overripe | 1 张 |

### 5.4 历史基线对比

| 版本 | 准确率 | 说明 |
|------|--------|------|
| 原始阈值 v0 | 5.00% (1/20) | `dark_ratio_threshold=0.18` 过低 |
| 当前阈值 v1 | 95.00% (19/20) | 组合规则 + 校准后阈值 |

### 5.5 错误分类样本

| 文件 | 真实 | 预测 | dark_ratio | yellow_ratio | contrast |
|------|------|------|------------|--------------|----------|
| `IMG_20260530_161901.jpg` | ripe | overripe | 0.6683 | 0.5019 | 0.8565 |

原因：该样本表面存在大量深色斑点或暗部，`dark_ratio` 较高且 `yellow_ratio` 接近阈值下界，因此触发组合规则。该样本在 ripe 与 overripe 边界上存在一定主观争议。

## 6. 当前版本局限性

1. 仅有 ripe 样本，无法验证三分类规则的整体可靠性。
2. `dark_ratio` 粒度较粗，无法区分斑点、茎端、尖端和阴影。
3. GLCM 基于主体区域边界框计算，背景填充可能引入轻微偏差。
4. HSV 阈值对光照、背景和拍摄角度敏感。
5. 当前分割流程默认单根香蕉场景，多根香蕉或复杂背景下可能失败。

---

## 7. 后续实验方向

1. 优先补充 `unripe` 和 `overripe` 样本，每类至少 20 张。
2. 补齐三类数据后重新统计特征分布，并重新校准规则阈值。
3. 增加暗斑连通区域数量、最大暗斑面积占比等细粒度特征。
4. 在复杂背景下测试分割鲁棒性，必要时引入 GrabCut 或颜色直方图反向投影。
5. 使用当前规则版作为 baseline，后续对比 SVM/KNN 分类效果。
